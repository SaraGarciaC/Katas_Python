## ✨ Descripción

Este repositorio contiene una serie de **katas de Python** resueltas
paso a paso en un Jupyter Notebook.\
Los ejercicios cubren desde operaciones básicas con listas y strings
hasta temas más avanzados como:

-   `map`, `filter`, `reduce` y `lambda`
-   Manejo de errores con `try/except`
-   Validaciones de datos
-   Diccionarios, sets y tuplas
-   Programación Orientada a Objetos (clases y métodos)
-   Pequeños programas interactivos con `input()`

Es ideal como **material de práctica**, **repaso** o **portfolio de
aprendizaje**.

------------------------------------------------------------------------

## 🧩 Contenido

Algunos ejemplos de katas incluidas:

-   Frecuencia de letras en un texto

## ▶️ Instalación y uso

### Opción 1: Jupyter Notebook

1.  Clona el repositorio:

    ``` bash
    git clone <URL_DEL_REPO>
    cd <NOMBRE_DEL_REPO>
    ```

2.  (Opcional) Crea un entorno virtual:

    ``` bash
    python -m venv .venv
    # Windows:
    .venv\Scripts\activate
    # macOS/Linux:
    source .venv/bin/activate
    ```

3.  Instala Jupyter:

    ``` bash
    pip install notebook
    ```

4.  Ejecuta:

    ``` bash
    jupyter notebook
    ```

5.  Abre el archivo:

        Katas_Python.ipynb

------------------------------------------------------------------------

### Opción 2: VS Code

-   Abre la carpeta del proyecto en VS Code
-   Instala las extensiones **Python** y **Jupyter**
-   Abre `Katas_Python.ipynb` y ejecuta las celdas
